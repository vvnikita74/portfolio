# Portfolio Landing

Статический сайт-портфолио без сборки. Главная страница: `index.html`.
Публикуемые ассеты лежат в `public/assets/` и имеют ASCII-имена для стабильной загрузки на GitHub Pages.

## Локальный запуск

```bash
./serve.sh
```

Откройте `http://127.0.0.1:8080/`.

## GitHub Pages

1. Загрузите репозиторий на GitHub.
2. Откройте `Settings -> Pages`.
3. В `Build and deployment` выберите `Deploy from a branch`.
4. Укажите ветку `main` и папку `/root`.
5. Сохраните настройки.
